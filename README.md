# NeoVim Config 

Heavily 'inspired' by '0 to LSP: Neovim RC From Scratch' by ThePrimeagen.

## Installation

Clone into /home/$USER/.config/nvim

## Features

Adds:
- Lazy plugin manager
- Telescope fuzzy finder
- Rose-pine colour scheme
- Treesitter highlighter
- Undotree
- LSP-Zero and Mason
- Custom remaps and settings
